# Thisisme

"This is me"

A clean and lightweight personal website template.

## Tech stack

- Vite
- React
- TypeScript

## Themes

This template supports multiple visual themes. See [THEMES.md](./THEMES.md) for the list of available themes.

Set the `VITE_THEME` environment variable to choose which theme to build:

```bash
VITE_THEME=default npm run build
```

During development, themes can be switched in the browser. A floating switcher appears at the bottom-right corner in dev mode.
